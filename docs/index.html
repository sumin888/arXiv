What it can do

📄

Full-text RAG

PDF downloaded and indexed on first visit. Hybrid vector + keyword search (SQLite-vec + FTS5) with Reciprocal Rank Fusion.

🔧

Tool-calling agent

Autonomously calls tools — search arXiv, fetch citations, explore GitHub repos, run experiment code — and cites chunk IDs in every answer.

🌉

Bridge paper discovery

Every few turns the agent finds a paper from a different domain using the same core method, with one-click cross-domain exploration.

⚗️

Experiment reproduction

Ask the agent to clone the paper's official GitHub repo, install dependencies, and run the code end-to-end to verify reported results.

🔀

Dual LLM provider

Works with Anthropic (Claude) or any OpenRouter model. Swap with one env var — no code changes.

🧩

Chrome side panel

Manifest V3 extension. Activates automatically on arxiv.org/abs/* — no copy-pasting URLs or switching tabs.

How it works

1

Navigate to any arXiv abstract page

The content script extracts the paper ID, title, authors, and abstract from the DOM and sends it to the side panel.

2

PDF is fetched and indexed

The backend downloads the PDF, splits it into 1200-character chunks, embeds them with all-MiniLM-L6-v2, and stores them in SQLite-vec. Subsequent visits are instant (already indexed).

3

Agent answers using retrieved context

Each query retrieves the top-10 chunks via hybrid search, injects them into the system prompt, and enters a tool-calling loop until a grounded answer is ready.

4

Bridge papers surface cross-domain connections

After every 3rd message the backend automatically finds a paper from a different field using the same method and appends a dismissible bridge card.

Quick setup

1 — Backend
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Create .env
echo "LLM_PROVIDER=openrouter
OPENROUTER_API_KEY=sk-or-..." > .env

uvicorn app.main:app --reload
2 — Load extension in Chrome
1. Open chrome://extensions
2. Enable Developer mode
3. Click "Load unpacked"
4. Select the repo root folder
5. Navigate to arxiv.org/abs/*
6. Click the extension icon

Full instructions in the README. Free OpenRouter API keys at openrouter.ai.
